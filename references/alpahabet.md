# Alphabet STÈLE — Les 25 Primitives

> Chaque primitive est un nœud ontologique indivisible. Trois faces co-définies :
> **glyphe** (forme iconique-indicielle), **mot-code** (nom symbolique-opératoire),
> **fonction** (ce que la primitive *fait* dans une chaîne).

L'alphabet est organisé en **4 strates**, lues comme couches de profondeur :

- **Strate I — Élémentaire** : les substances (ce qui est)
- **Strate II — Opérative** : les verbes (ce qui agit)
- **Strate III — Modale** : les modulateurs (ce qui colore)
- **Strate IV — Méta** : les bornes (ce qui structure l'énoncé)

---

## STRATE I — ÉLÉMENTAIRE (substances ontologiques, 8 signes)

Les substances sont les *sujets* possibles d'un énoncé STÈLE. Toute chaîne performative a au moins une substance.

### `⊙` SOURCE
- **Fonction** : Désigne l'origine, le point d'émergence, ce d'où une dynamique procède.
- **Pragmatique** : Énoncer SOURCE oriente l'attention vers la cause-mère, le germe.
- **Contraires fréquents** : TRACE (l'origine vs ce qui en reste), VIDE (origine plénière vs origine vide).
- **Exemple** : `⊙∿` = une source en mouvement (quelque chose émerge et coule).

### `◯` FORME
- **Fonction** : Ce qui est advenu, la manifestation, le contour stabilisé.
- **Pragmatique** : Marque qu'une dynamique s'est cristallisée en quelque chose de saisissable.
- **Contraires fréquents** : VIDE (manifesté vs non-manifesté), MOUVEMENT (figé vs en cours).
- **Exemple** : `∿◯` = un mouvement qui se fige en forme.

### `Ø` VIDE
- **Fonction** : Absence, espace ouvert, non-encore-advenu, blanc créateur.
- **Pragmatique** : Le VIDE en STÈLE n'est pas le néant : c'est l'ouvert qui appelle.
- **Contraires fréquents** : FORME (vide vs plein), SOURCE (vide vs émergence).
- **Exemple** : `Ø⊙` = du vide émerge une source (la naissance pure).

### `∿` MOUVEMENT
- **Fonction** : Flux, transformation en cours, devenir, dynamique active.
- **Pragmatique** : Signale qu'on est en transit, pas en arrivée.
- **Contraires fréquents** : LIMITE (mouvement vs arrêt), FORME (devenir vs advenu).
- **Exemple** : `◯∿◯` = une forme se transforme en une autre forme.

### `⊥` LIMITE
- **Fonction** : Frontière, bord, ce qui termine, ce qui sépare.
- **Pragmatique** : Marque un seuil, une coupure, une fin de domaine.
- **Contraires fréquents** : MOUVEMENT (limite vs continuité), AXE (frontière vs colonne).
- **Exemple** : `∿⊥` = un mouvement qui bute sur une limite.

### `✦` AXE
- **Fonction** : Colonne vertébrale, structure portante, ce qui tient sous le flux.
- **Pragmatique** : Désigne la stabilité interne, le pivot autour duquel tout tourne.
- **Contraires fréquents** : MOUVEMENT (stable vs en cours), VIDE (porteur vs ouvert).
- **Exemple** : `✦↻` = un axe en cycle (une structure qui se répète sur elle-même).

### `◊` TRACE
- **Fonction** : Empreinte, mémoire, ce qui reste après que la chose a passé.
- **Pragmatique** : Indique la rémanence, le résidu signifiant.
- **Contraires fréquents** : SOURCE (trace vs origine), VIDE (résidu vs ouvert).
- **Exemple** : `⊙◊` = la source a laissé une trace (mémoire d'origine).

### `⊛` NOEUD
- **Fonction** : Point de complexité, croisement, là où plusieurs dynamiques convergent.
- **Pragmatique** : Marque la densité, la complication, le lieu où ça se noue.
- **Contraires fréquents** : AXE (complexe vs simple), VIDE (densité vs ouvert).
- **Exemple** : `∿∿⊛` = deux mouvements convergent en un nœud.

## STRATE II — OPÉRATIVE (verbes performatifs, 7 signes)

Les opératives sont les *verbes*. Elles **agissent** sur les substances qui les précèdent ou suivent. Une chaîne performative a au moins une opérative.

### `⟁` FRACTURER
- **Fonction** : Ouvrir, fissurer, briser un scellé, rompre une continuité.
- **Pragmatique performative** : Énoncer FRACTURER agit comme une question déstabilisante, une fissure consciente.
- **Régime** : Acte d'ouverture. À utiliser quand quelque chose doit céder.
- **Exemple** : `⊛⟁` = le nœud est fracturé (la complication s'ouvre).

### `⊗` LIER
- **Fonction** : Connecter, tisser, mettre en relation, ponter.
- **Pragmatique performative** : Énoncer LIER crée un pont actif entre deux entités.
- **Régime** : Acte de tissage. À utiliser quand quelque chose doit se relier.
- **Exemple** : `⊙⊗◊` = la source est liée à sa trace (lignage assumé).

### `✶` RÉVÉLER
- **Fonction** : Porter au visible, dévoiler, faire apparaître ce qui était caché.
- **Pragmatique performative** : Énoncer RÉVÉLER met sous lumière sans imposer d'interprétation.
- **Régime** : Acte de dévoilement. À utiliser pour nommer sans définir.
- **Exemple** : `Ø✶◯` = du vide on révèle une forme (manifestation).

### `⊜` SCELLER
- **Fonction** : Clore, finir, mettre le point final, fermer un cycle.
- **Pragmatique performative** : Énoncer SCELLER ferme un dossier, achève.
- **Régime** : Acte de clôture. À utiliser quand quelque chose doit finir.
- **Exemple** : `◯⊜` = la forme est scellée (clôture).

### `⌒` PLIER
- **Fonction** : Replier, intérioriser, mettre en réserve, garder en latence.
- **Pragmatique performative** : Énoncer PLIER met de côté sans détruire ; mise en germe.
- **Régime** : Acte de réserve. À utiliser quand quelque chose doit attendre.
- **Exemple** : `⊙⌒` = la source est pliée (mise en germe).

### `⥀` INVERSER
- **Fonction** : Retourner, faire pivoter, lire à l'envers.
- **Pragmatique performative** : Énoncer INVERSER force un changement de perspective.
- **Régime** : Acte de retournement. À utiliser pour basculer le point de vue.
- **Exemple** : `⊥⥀` = la limite est inversée (ce qui fermait ouvre).

### `⟶` TRANSMETTRE
- **Fonction** : Passer, donner, faire circuler vers un autre lieu/sujet/temps.
- **Pragmatique performative** : Énoncer TRANSMETTRE déplace activement.
- **Régime** : Acte de don. À utiliser pour la circulation, l'héritage, la passation.
- **Exemple** : `◊⟶⊙` = une trace est transmise jusqu'à devenir source (relais d'origine).

## STRATE III — MODALE (modulateurs, 6 signes)

Les modales **modifient le signe qui les précède immédiatement**. Elles ne peuvent pas tenir seules.

### `↻` CYCLE
- **Fonction** : Indique la répétition, le retour, la circularité temporelle.
- **Application** : Modifie une substance ou une opérative pour signaler récurrence.
- **Exemple** : `∿↻` = mouvement en cycle (devenir cyclique).

### `⌖` LIEU
- **Fonction** : Ancre dans une situation spatiale ou contextuelle précise ; « ici ».
- **Application** : Modifie une substance pour la situer.
- **Exemple** : `⊛⌖` = nœud situé (complication localisée, pas abstraite).

### `▲` INTENSITÉ
- **Fonction** : Amplifie le signe qui précède ; charge, force, urgence.
- **Application** : Modifie n'importe quel signe pour signaler une amplitude marquée.
- **Exemple** : `⟁▲` = fracture intense (rupture majeure, pas simple fissure).

### `↺` RÉFLEXIVITÉ
- **Fonction** : Indique que le signe précédent s'applique à lui-même ; auto-référence.
- **Application** : Modifie une opérative pour la rendre auto-appliquée.
- **Exemple** : `✶↺` = révéler appliqué à soi (auto-dévoilement).

### `⊘` NÉGATION
- **Fonction** : Nie, refuse, exclut le signe qui précède.
- **Application** : Modifie un signe pour signaler son absence ou son refus.
- **Exemple** : `⊜⊘` = non-scellé (refus de clôture).

### `◐` CONDITIONNEL
- **Fonction** : Indique latence, possibilité, état conditionnel ; « peut-être ».
- **Application** : Modifie un signe pour le mettre au mode possible.
- **Exemple** : `⊙◐` = source possible (potentialité d'origine, pas encore advenue).

## STRATE IV — MÉTA (bornes structurelles, 4 signes)

Les méta **encadrent** l'énoncé ou désignent l'énoncé lui-même. Ce sont les signes de ponctuation performative.

### `⟦` OUVERTURE
- **Fonction** : Marque le début d'un énoncé performatif fort.
- **Usage** : Toujours en première position. Indique que ce qui suit est un acte, pas une description.
- **Exemple** : `⟦⊙⟁◯⟧` = j'ouvre : la source est fracturée vers la forme : je clos.

### `⟧` FERMETURE
- **Fonction** : Marque la fin d'un énoncé performatif fort.
- **Usage** : Toujours en dernière position, en appariement avec `⟦`.
- **Exemple** : voir ci-dessus.

### `◉` AUTO
- **Fonction** : Auto-désignation — le signe se montre lui-même comme énoncé.
- **Usage** : Marque un méta-énoncé (« je dis ce que je dis »).
- **Exemple** : `◉✶` = l'énoncé se révèle comme énoncé (signature consciente).

### `◌` SILENCE
- **Fonction** : Non-dit assumé, blanc volontaire, ce qu'on ne nomme pas.
- **Usage** : Marque la présence d'un creux dans la chaîne. À ne pas confondre avec VIDE (substance) — SILENCE est une *position* dans l'énoncé.
- **Exemple** : `⊙∿◌` = une source en mouvement vers du silence (l'inachevable assumé).

## Notes d'usage

**Sur le choix iconique.** Les glyphes ont été choisis pour leur ressemblance avec ce qu'ils nomment : `⊙` (un point dans un cercle = germe contenu), `∿` (sinusoïde = onde), `⟁` (triangle dans un triangle = cassure imbriquée), `⊛` (cercle avec branches = convergence). Ce n'est pas un système purement arbitraire.

**Sur l'extension.** L'alphabet est volontairement clos à 25 primitives. Pour enrichir, ne pas ajouter de signe : utiliser plutôt les **combinaisons**. La richesse réside dans la combinatoire, pas dans l'extension du lexique.

**Sur l'usage en sortie.** Quand Claude produit une chaîne STÈLE, toujours fournir aussi la **transcription verbale** (mots-codes séparés par des points). Exemple : `⟦⊙⟁◯⟧` → `OUVERTURE.SOURCE.FRACTURER.FORME.FERMETURE`. Cela garantit la lisibilité même hors environnement Unicode-compatible.
